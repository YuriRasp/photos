# photo - программа для редактирования фото

короче простенькая программа для редактирования приложения

## полный функционал приложения:

1. Поворот фото влево или вправо на 90 градусов
2. Отзеркаливание фото
3. Увеличение резкости
(сразу минус: можно только повысить резкость. и при достижении максимально возможного значения приложение зависает, простите)
4. Сделать фото чернобелым
5. Сброс всех изменений

### P.s.

К сожалению эта программа запускается только через запуск кода в VC Code(другие программы для кода я не запускал, т.к. попросту нету),
так же вам нужно будет скачать через терминал PyQt5, Pillow и может быть Image, я не знаю.
